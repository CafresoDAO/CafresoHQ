# 2026-04-21 — Session 1: Bootstrap

First research session for the ICP vault. Goal was to lay a foundation of atomic concept notes covering the minimal set a dev needs before they can meaningfully read deeper material, plus one runnable tutorial to anchor the concepts.

## What was studied

Eight atomic concepts, each drafted from the official DFINITY docs / GitHub:

- [[canisters]] — Wasm module + state + identity + cycles wallet as one addressable object.
- [[cycles]] — reverse-gas model; stable XDR peg; per-subnet pricing.
- [[subnets]] — ICP's sharding unit; a BLS-signed replicated state machine.
- [[principals]] — single identity type covering users, canisters, and anonymous callers.
- [[motoko]] — actor-language designed around canister semantics; orthogonal persistence + `stable var`.
- [[rust-cdk]] — `ic-cdk` + `ic-cdk-macros` + `ic-stable-structures` as the production workhorse stack.
- [[dfx]] — the CLI that ties build, local replica, identity, and deploy together.
- [[internet-identity]] — WebAuthn passkeys + per-origin principal derivation.

And one tutorial: [[hello-canister-motoko]] — the canonical `dfx new` → `dfx start` → `dfx deploy` → `dfx canister call greet '("world")'` loop.
