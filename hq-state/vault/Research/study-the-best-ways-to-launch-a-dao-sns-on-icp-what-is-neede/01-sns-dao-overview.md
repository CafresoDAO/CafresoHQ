# SNS DAO on ICP — Overview

## What is an SNS?

A **Service Nervous System (SNS)** is ICP's built-in DAO framework that lets dapp developers hand full ownership and governance of their project to a decentralized community. It is provided natively by the Internet Computer platform — no custom smart contracts needed.

## Core Functions

- Raises initial funds via a **decentralized swap** (similar to an IDO/token sale)
- Transfers dapp canister ownership to the SNS governance canister
- Community holds **neurons** (staked governance tokens) and votes on proposals:
  - Canister upgrades
  - Treasury spend
  - Tokenomics changes
  - General project direction

## Key Milestones (2023–2025)

- **2023** — dubbed "The Year of SNS DAOs"; first wave of live SNS projects
- **2024** — DFINITY expanded SNS capabilities, improved swap mechanics, and added more developer tooling
- Active DAOs tracked live at: https://dashboard.internetcomputer.org/sns

## Why SNS vs. Rolling Your Own DAO

| Factor | SNS | Custom DAO |
|---|---|---|
| Time to launch | Faster (template) | Slower |
| Security | Audited by DFINITY | Self-audited |
| NNS integration | Native | Manual |
| Flexibility | Limited by framework | Full |
