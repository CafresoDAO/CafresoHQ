# Cafreso: Project Overview and Tech Stack

Based on the project's main README, here is the initial high-level summary:

*   **Concept:** A coffee subscription service that integrates blockchain features. It allows users to stake tokens and earn rewards.
*   **Core Value Prop:** Blends a physical product (coffee) with DeFi elements (staking/rewards).

## Technology Stack

The project is built on the Internet Computer Protocol (ICP).

*   **Backend:** Motoko, utilizing CanDB for data persistence. This is a key detail, especially as we are planning our own CanDB update.
*   **Frontend:** Svelte-Kit, with JavaScript and TypeScript.
*   **Key Config Files:** The repository includes standard ICP project files like `dfx.json`, `canister_ids.json`, and `mops.toml`, confirming the development and deployment environment.
